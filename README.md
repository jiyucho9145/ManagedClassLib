# ManagedClassLib
ManagedClassLib is a C language class library which consists of several managed classes to make a small CUI tool.

## Requirement
- OS: Linux or macOS
- gcc
- make

## Installation
1. Clone this repository.
2. Move `bin` directory.
3. Run `make` command.
4. Run `./greeting` program.
